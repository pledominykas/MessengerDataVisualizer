# MessengerDataVisualizer
MessengerDataVisualizer is a WPF application for visualizing your facebook data.

## Presentation & usage
 1. Download your facebook information by following  [these instructions](https://www.facebook.com/help/212802592074644) (currently you only need to download comment, friend, message and profile information).
 2. Extract the files using WinRar, 7zip or other software.
 3. Drag and drop your information to MessengerDataVisualizer.
 
 ![](https://media.giphy.com/media/mCz8lbkp7pSdzkUC7b/giphy.gif)
 
 4. Explore your data.
